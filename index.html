Program Keahlian Teknik Komputer dan Jaringan

Teknik Komputer dan Jaringan (TKJ) adalah program unggulan yang melatih siswa di bidang teknologi informasi dan jaringan komputer.

Instalasi dan Konfigurasi Jaringan

Mengajarkan siswa untuk menginstal dan mengonfigurasi jaringan lokal (LAN), wireless, dan jaringan berbasis cloud.

Keamanan Jaringan

Pelatihan mendalam tentang keamanan jaringan, firewall, dan VPN untuk melindungi data serta mencegah serangan siber.

Administrasi Server

Mempelajari manajemen server dengan berbagai sistem operasi seperti Windows Server dan Linux.

Pemrograman Dasar

Siswa mendapatkan dasar-dasar pemrograman, yang bermanfaat dalam pengembangan perangkat lunak dan aplikasi jaringan.

Manajemen Proyek IT

Pelatihan untuk mengelola proyek berbasis teknologi informasi.
